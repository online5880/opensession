Quickstart

npx @online5880/opensession init
npx @online5880/opensession sync --project demo
npx @online5880/opensession start --project-key demo --actor mane
npx @online5880/opensession status --project-key demo
npx @online5880/opensession log --limit 50
npx @online5880/opensession ops --project-key demo

Commands

  • init: validate config + Supabase connectivity
  • sync --project demo: sync local/remote state
  • start --project-key demo --actor mane: open a live session
  • resume --session-id <id> --actor mane: resume prior session
  • viewer --host 127.0.0.1 --port 5880: run web viewer
  • report --project-key demo --json: KPI + trend report

Troubleshooting

  • PGRST205 (table not found): apply schema SQL once, then rerun init and sync.
  • Missing project key: run sync --project demo or pass --project-key demo.
  • Auth/network errors: confirm Supabase URL format and publishable key prefix.
  • No active sessions in viewer: verify latest run used the same project key and actor scope.
